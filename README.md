# Repositório para Formação CSS Web Developer

## Objetivos

- Armazenar códigos de projetos de forma ordenada dentro deste repositório;
- Consolidar conhecimentos aprendidos através de projetos práticos.

## Organização

- Para cada desafio proposto na formação haverá um documento no repositório com o nome do desafio visando o cumprimento das atividades da DIO;
- A pasta de Assets será compartilhada com todos os "projetos" armazenando os arquivos específicos;
- Utilização de um repositório único para toda a formação.
